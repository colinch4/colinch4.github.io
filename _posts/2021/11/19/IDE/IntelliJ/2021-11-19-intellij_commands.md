## IntelliJ Commands
IntelliJ 명령어 모음  
Key Map : macOS

<code>⌘</code>  Command  
<code>⌃</code>  Control  
<code>⌥</code>  Option  

⌨️ Use your keyboard more, and your mouse less

> 링크를 클릭하여 이동

## C
  * Close
    * Close Tab : ```⌘ + w```  
    현재 열려 있는 Tab 닫기


## D
  * Delete
    * Safe Delete : ```⌘ + delete``` 


## E
  * Extract  
    * Extract Field : ```⌥ + ⌘ + f```
    * Extract Method : ```⌥ + ⌘ + m```
    * Extract Variable : ```⌥ + ⌘ + v```

## N
  * Navigate
    * Navigate Back :  ```⌘ + [```
    * Navigate Front : ```⌘ + ]```


## R
  * Refactor : ```⌃ + T```
    커서 부분 Refactor 

## S
  * Show Parameter : ```⌘ + p```
    파라미터 정보 보여주기 

  * Show Toolbar
    * Porject : ```⌘ + 1```
    * Favorites : ```⌘ + 2```
        * Bookmarks
        * Breakpoints
    * Run Console : ```⌘ + 4```
    * TODO : ```⌘ + 6```
    * Structure : ```⌘ + 7```
    * Service : ```⌘ + 8```
    * Version Control : ```⌘ + 9```
