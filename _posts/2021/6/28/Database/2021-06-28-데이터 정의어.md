## 데이터 정의어

- 데이터 정의어의 정의

  - 데이터베이스를 정의하거나 수정할 목적으로 사용하는 언어

- 데이터 정의어의 특징

      데이터베이스 스키마를 컴퓨터가 이해할 수 있도록 기술하는데 사용

      논리적 데이터 구조를 정의(스키마, 외부 스키마 기술)

      물리적 데이터 구조를 정의(내부 스키마 기술)

      Create, Drop, Alter
