# eclipse 단축키

- **syso + ctrl + space** : System.out.println
- **ctrl + D** : 라인 삭제
- **ctrl + alt** + 방향키 : 라인 복사
- **alt + 방향키** : 라인 이동
- **ctrl + shift + F** : 코드 정리
- **f11**   : 컴파일
- **shift + enter** : 아랫 줄 엔터
- **ctrl + shift + enter** : 윗 줄 엔터
- **ctrl + shift +  o** : import 선언
