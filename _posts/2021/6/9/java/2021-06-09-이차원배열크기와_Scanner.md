## 이차원배열크기와 Scanner

### 이차원 배열 크기 구하기

* 행 : score.length
* 열 : score[i].length



### Scanner 

* 입력받는 역활의 API

  ```java
  Scanner sc = new Scanner(System.in);  //키보드로부터 입력을 받겠다.
  
  sc.next();                 // 단어 (워드), String ----->Space
  sc.nextLine();			   // ------> Enter  
  
  //숫자
  sc.nextInt();             // int
  sc.nextDouble();           // double
  
  String s = sc.next();
  sc.nextLine();   //번가락하면서 사용하려면 한번 sc.nextLine()을 통하여 비워준다.
  String s1 = sc.nextLine();
  ```
