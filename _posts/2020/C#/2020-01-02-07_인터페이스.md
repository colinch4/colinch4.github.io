## 인터페이스

#### 인터페이스 특징

* 인터페이스는 구현 없이 형식만 포함
* 다중 상속 가능
* 상속으로만 사용하고 생성은 불가능
* 인덱스, 인덱서, 메서드, 속성을 포함
* 기본 권한은 `public`



#### 형식

> ```c#
> interface I인터페이스명 //관례적으로 클래스명 앞에 'I'를 붙임
> {
>     …
> }
> ```

#### 

#### 다중 상속의 예

> ```c#
> class C : IA, IB
> {
>     …
> }
> ```



## partial

####  partial

* 클래스나 구조체, 인터페이스, 메서드의 정의를 둘 이상의 소스 파일로 분할
* 규모가 큰 프로젝트일 때 하나의 파일에 다수가 접근할 때 사용
